# Orbit AI - Python Bindings

Python bindings for Orbit AI with LangChain integration, built with PyO3.

**Current Status**: Basic PyO3 setup complete. The bindings compile and basic functions are accessible from Python. However, full LangChain integration requires a Tauri AppHandle, which limits standalone Python usage.

## Building

### Prerequisites

- Rust toolchain (1.75 or later)
- Python 3.9 or later
- maturin (`pip install maturin`)

### Build the Python Extension

```bash
# From the orbit-ai crate directory
cd crates/orbit-ai

# Build and install in development mode
maturin develop

# Build a wheel for distribution
maturin build --release
```

## Usage

### Basic Example

```python
import asyncio
from orbit_ai import LangChainBot, ScreenshotService

async def main():
    # Initialize with API key
    bot = LangChainBot(api_key="your-openrouter-api-key")

    # Send a chat message
    response = await bot.chat("Hello, what is Orbit?")
    print(response)

    # Clear conversation memory
    await bot.clear_memory()

    # Capture a screenshot
    screenshot = ScreenshotService()
    image_base64 = await screenshot.capture()

asyncio.run(main())
```

### Environment Variables

- `OPENROUTER_API_KEY`: Your OpenRouter API key for LLM access

### Python API

#### LangChainBot

- `__init__(api_key=None)`: Initialize the bot with optional API key
- `async chat(message: str) -> str`: Send a message and get a response
- `async stream_chat(message: str) -> str`: Send a message and stream the response
- `async clear_memory()`: Clear the conversation memory

#### ScreenshotService

- `__init__()`: Initialize the screenshot service
- `async capture() -> str`: Capture a screenshot and return as base64

### Low-Level API

The module also exports the raw PyO3 bindings:

- `PyLangChainBot`: Direct Rust struct wrapper
- `PyScreenshotService`: Direct screenshot service wrapper
- `create_langchain_bot()`: Async factory function
- `set_openrouter_api_key(key)`: Set the API key
- `get_system_prompt()`: Get the system prompt

## Integration with LangChain

This module integrates with the Rust-based langchain-rust library and provides Python bindings for seamless use in Python applications. It supports:

- OpenRouter LLM integration
- Tool calling and agents
- Memory management
- Screenshot capture tools
- Streaming responses
